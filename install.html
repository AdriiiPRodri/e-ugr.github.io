Guía de instalación

Añade {{ site.title }} a tu pantalla de inicio para que funcione como una app más

Android

  1. Abre esta página en Chrome o Chromium
  2. Abre el menú
  3. Selecciona Añadir a pantalla de inicio
  4. Pulsa Añadir

iOS

  1. Abre esta página en Safari
  2. Pulsa Compartir (rectángulo con flecha hacia arriba)
  3. Pulsa Añadir a pantalla de inicio
